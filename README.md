# 1. منصةالتعليمية للأطفال (KidsFlow Learning Platform)
باستخدام إطار العمل CodeIgniter 4 (PHP).
تستخدم قاعدة بيانات MySQL لتخزين البيانات.
تقسيم الملفات MVC

تعتمد المنصة على نظام إدارة تعليمي موجه لثلاث فئات من المستخدمين 

المسؤول (Admin): يتولى إدارة المنصة بشكل كامل، بما في ذلك إضافة وتعديل المواد الدراسية، الدروس، القصص التعليمية، والألعاب، بالإضافة إلى إدارة حسابات المستخدمين.
ولي الأمر (Parent): يمكنه إضافة حسابات لأطفاله، متابعة تطورهم الدراسي، عرض تقدمهم في الدروس، وإضافة ملاحظات أو تحديث مستوياتهم التعليمية.
الطفل (Child): هو المستخدم النهائي الذي يتفاعل مع المحتوى التعليمي من خلال مستويات متدرجة، دروس تفاعلية، قصص، وألعاب تعليمية تهدف إلى تحبيبه في التعلم وكسب النقاط.
أهم ميزات النظام:

تقسيم المحتوى إلى مستويات ومواد دراسية.
نظام تتبع التقدم (Progress Tracking).
محتوى تعليمي متنوع (دروس، قصص، ألعاب).
لوحات تحكم مخصصة لكل نوع من المستخدمين.
